Features Cellipede pipulate

universal access to functions
       you can use your functions from anywhere because the app can be used as a service provider

google-like spreadsheets via JQuery and Handsontable
       you are not locked in to using only Google's spreadsheets

fast and reliable

easily install on Heroku and run it for free
       let Heroku worry about setup, configuration, scaling, and security

well designed, coded, and implemented
       it's great as a guide for learning python and flask

no remote dependencies (like Google Sheets API)

a free and open source python app

lots of pre-built SEO functions

capable of running on Tiny Core via QEMU or VirtualBox  *

* Notes:
  • pipulate tries to force you to use levinux, which is nothing more than Tiny Core with QEMU and several bash scripts
  • to learn how to install software, it's best to just download Tiny Core and QEMU then manually install the software
  • even better, because Tiny Core is not exactly like most linux distro's, install Ubuntu in a VirtualBox vm and learn skills that will probably last a lifetime
