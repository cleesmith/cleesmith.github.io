Encoded Prompt Decoder

Paste a base64 + gzip string (e.g., Novelcrafter exports that start with H4sIA…). I’ll decode, gunzip, and show JSON/text. If it matches Novelcrafter’s shape, I’ll also extract a clean “system prompt” for easy copy/paste.

Tip: Works best in modern Chromium browsers (uses DecompressionStream('gzip')).
