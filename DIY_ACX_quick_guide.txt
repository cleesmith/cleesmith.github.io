
                     --- Overview ---
DIY recording/uploading of audio/MP3 files to ACX (audible), in short:
ACX recordings with no "Audio Analysis" issues

1. install Audacity ... mine is Mac version 3.1.3

2. add two Nyquist plug-ins: (a two step process for each plug-in)
   step 1:
	Tools > Nyquist Plug-in Installer
	in popup box click Browse... > 
	then select ".ny" file downloaded
	Allow overwriting: Disallow
	(1) ACX-Check.ny
	(2) DeClicker.ny
   step 2:
	Effect > Add/Remove Plug-ins... = Manage Plug-ins popup box
	highlight each plug-in to set it's State to Enabled
	click OK	

3. Effect > Compressor > Manage > Import = to load compressor text file for Rode NT1 mic

4. Tools > Macros... > Import... = to load text of commands to run, the macro file

-----------------------------------------------------------------------------------
... note: the above 4 steps are only done once at initial installation, after that, 
	   you repeat the following 6 steps for each audio/MP3 recording
-----------------------------------------------------------------------------------

5. record a chapter/whatever with 1 second of room noise at the beginning/end

6. stop/rewind then click on track name/title bar to select the entire track

7. Tools > Apply Macro > select macro from the list, this may run for a while

8. ACX Check will popup the results, if all looks good now export it as MP3

9. File > Export > Export as MP3 > name_it_whatever.mp3, select folder, ensure:
	File type: MP3 Files
	Bit Rate Mode: Constant
	Quality 192 kbps
	Channel Mode: Force export to mono
	click Save, which may take some time to save

10. upload to your ACX project and wait for "Audio Analysis"
