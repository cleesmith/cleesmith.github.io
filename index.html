  • Tiny Core with Python plus

    A pre-installed Python development environment in Tiny Core for use on Windows with QEMU or VirtualBox.
    This includes: python/pip, compile tools, openssh, vim/nano, and git.
    It is a perfect way to get your feet wet and begin learning Linux/Python, or just use it to explore Cellipede ... all without suffering through installing all of that software.

March 31, 2015

cellipede can be used as a service for all functions:

use it in a google spreadsheet like: 

      =IMPORTHTML("http://cellipede.herokuapp.com/sheet_function?afunction=tweets&url=http://www.amazon.com/","table",1)
or use it in another program, like curl: 
      curl "http://cellipede.herokuapp.com/sheet_function?afunction=tweets&url=http://www.amazon.com/"
or use it directly in a web browser: 
      http://cellipede.herokuapp.com/sheet_function?afunction=tweets&url=http://www.amazon.com/
      or:
      http://cellipede.herokuapp.com/sheet_as_table?id=1
this feature really opens up cellipede giving you access to all of the functions from just about anywhere
